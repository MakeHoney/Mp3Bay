## Requirements

#### Make config.js

What configuration file looks like under /src

``````javascript
export default {
  API_HOST: 'http://localhost:8888',
  LOCAL_PROVIDER: 'http://localhost:8545',
  HOST: 'http://localhost:8081'
}
``````
